# Introduce Parameter Object Tutorial

Code-free interactive web tutorial for Martin Fowler's **Introduce Parameter Object** refactoring.

The learning surface uses visual app metaphors instead of code snippets:

- delivery app conditions
- map route conditions
- music recommendation conditions
- payment conditions

## Commands

```powershell
npm.cmd install
npm.cmd run dev
npm.cmd run build
npm.cmd run test
npm.cmd run lint
```

## Verification

- Vite production build
- Vitest coverage
- Biome + Stylelint
- Playwright visual/E2E checks
- Lighthouse performance check
- Semgrep security scan
